Background

  • Business: Delta Hand Pies
  • Problems to solve:
    • The client is manually processing all orders each week
    • Having to map out delivery routes of orders to drivers.
    • There is a lack of flow in the delivery process.
    • Drivers currently do not have an app they can rely on for order information, delivery address, and instructions.

Scope

  • This project is for the creation of a data management application for the client’s business of making custom pies and delivering it directly to a customer’s door.
  • The application will allow operation managers to sort, manage, and assign delivery data to specific drivers.
  • There will be a corresponding mobile application whereby anyone using the app can view and manage detailed order information.
  • Able to automate the process of notifying customers regarding their order status.

Need

  • The client needs an automation of weekly delivery data management for Delta Hand Pies.
  • The system would import raw sales data via a pre-built WordPress - WooCommerce through APIs.
  • There is a need to be able to manipulate and express these data in a useful way.
  • Delivery drivers would be able to look up the assigned information via an application.
  • The application would output the delivery information in a JSON format.
  • The tool would also be used by operation managers to check for pending and completed deliveries.
  • The client needs a quick and efficient method for communication between delivery driver and the customers.
Driver Delivery

Proposed Solution

  • Build a SQL internal database to fetch data from a built WordPress-WooCommerce through APIs.
  • Allowing the data to be manage on an internal website.
  • An accompanying driver’s app will display detailed order information.
  • An automated messaging system using Twilio APIs for communication between drivers to the customers will be implemented into the drivers app.

Highlights

  • We seek to lower the client’s dependency on third party eCommerce application.
  • The project will provide us a wide variety of skillsets.
